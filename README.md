# 🧪 QA Automation - Karate DSL (ServeRest API)

## 📌 Descripción

Proyecto de automatización de pruebas para la API de usuarios de ServeRest utilizando Karate DSL.

## 🛠 Tecnologías

* Java
* Maven
* Karate DSL

## ⚙️ Configuración

### Prerrequisitos

* Java 11 o superior
* Maven

## ▶️ Ejecución

```bash
mvn clean test
```

## 📂 Estructura del proyecto

* `features/`: escenarios de prueba
* `helpers/`: utilidades (data, schemas)
* `runners/`: ejecución de tests
